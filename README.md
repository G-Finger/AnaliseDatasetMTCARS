# AnaliseDatasetMTCARS
Análise acadêmica e didática do dataset mtcars em linguagem R. 
As bibliotecas necessárias estão descritas no programa e devem ser observadas para que o programa rode de maneira correta.
-> Instalar as bibliotecas e carregar no ambiente, como no exemplo.
      library("nome da bibliotecas")
Lembrar que o símbolo "#" significa comentário no programa e não é "lido" na execução. 
Existem dois arquivos neste repositório: parte I e parte II. Ambas explicações estão no canal do professor no youtube - "Matematicando @prof.gustavofinger"
O programa é público e pode ser usado de forma gratuita desde que citada a fonte. @prof.gustavofinger.
